## ✨ Highlights

- 🎨 **Glassmorphic dark theme** with animated gradient backgrounds
- ⚡ **Static export** via Next.js — fast, SEO-optimized, no server needed
- 📱 **Fully responsive** — optimized for all screen sizes
- ♿ **Accessible** — WCAG 2.1 AA compliant with keyboard navigation, ARIA labels, and semantic HTML
- 📊 **Scroll progress bar** and smooth section transitions
- ✍️ **Integrated blog** with MDX support
- 📄 **Downloadable resume** with a single click

## 🛠️ Tech Stack

| Layer | Technology |

## 🚀 Getting Started

```bash
# Clone
git clone https://github.com/himanshu31shr/portfolio.git
cd portfolio

# Install
npm install

# Dev server
npm run dev

# Build for production
npm run build
```
